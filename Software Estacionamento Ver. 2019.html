Como Instalar / Remover

O exemplo abaixo se aplica para qualquer um dos nossos softwares.

O software registra em seu equipamento bibliotecas e componentes necessários para a execução. Todos esses arquivos são removidos do computador caso o mesmo seja desinstalado.

**********

Localize o software desejado na primeira página do site, clique em "saiba mais +" em seguida clique no botão INSTALAR SOFTWARE.

01 - Assim que o download concluir execute o aplicativo.

02 - Na tela de aviso de segurança clique em Executar.

03 - O processo de instalação é simples e transparente para que o usuário não tenha dificuldades na conclusão.

Caso tenha alguma dúvida, entre em contato conosco. Teremos o prazer em auxiliá-lo.

**********

Processo para Desinstalação

01 - Abra o Painel de Controle do Windows.

02 - Selecione adicionar/remover programas.

03 - Localize facilmente o software pelo nome e desinstale.

03 - O processo de desinstalação removerá todos os componentes instalados.
